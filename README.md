# Final Project - Emotion Detector

Summary:
This application showcases the use and/or knowledge of requests, flasks, and all included libraries
that were introduced in Course-9 of the Full Stack Developer Course offered in Coursera by IBM.
The application represents simple emotional analysis given statement, that communicates with the package
module's function to call an API to return a score based off of that response. The response is then returned
to the client, displaying emotions as well as their corresponding emotional scores. Summarizing a dominant
emotion towards the end.
